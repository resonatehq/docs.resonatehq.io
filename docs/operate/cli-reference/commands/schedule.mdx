Manage schedules in the Resonate system.

```shell title="Schedule commands"
resonate schedules <subcommand> [args] [flags]
```

## Available subcommands

- `create` - Create a new recurring schedule
- `get` - Retrieve schedule details
- `search` - Search for schedules by ID pattern
- `delete` - Remove a schedule

## Quick examples

```shell title="Create hourly schedule"
resonate schedules create hourly-sync \
  --cron "0 * * * *" \
  --promise-id "sync-{{.timestamp}}" \
  --description "Hourly data synchronization"
```

```shell title="Create daily schedule"
resonate schedules create daily-cleanup \
  --cron "0 2 * * *" \
  --promise-id "cleanup-{{.timestamp}}" \
  --promise-timeout 1h
```

```shell title="List all schedules"
resonate schedules search "*"
```

```shell title="Delete a schedule"
resonate schedules delete hourly-sync
```

## Learn more

Run `resonate schedules --help` to see all subcommands.

Run `resonate schedules <subcommand> --help` for detailed usage.
