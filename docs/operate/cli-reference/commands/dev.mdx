Start the Resonate server with sensible defaults for local development.

```shell title="Start development server"
resonate dev [flags]
```

This command is an alias for:

```shell
resonate serve --aio-store-sqlite-path ':memory:' --aio-store-postgres-query 'sslmode=disable'
```

Uses in-memory SQLite storage and disables PostgreSQL SSL for easy local development.

## Quick example

```shell title="Start server on default ports"
resonate dev
```

```shell title="Use persistent SQLite file"
resonate dev --aio-store-sqlite-path ./dev.db
```

```shell title="Run on a different port"
resonate dev --api-http-addr :9000
```

## Learn more

Run `resonate dev --help` for all available flags and options.
