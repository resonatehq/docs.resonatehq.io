Manage Durable Promises in the Resonate system.

```shell title="Promise commands"
resonate promises <subcommand> [args] [flags]
```

## Available subcommands

- `create` - Create a new Durable Promise
- `resolve` - Resolve a promise with data
- `reject` - Reject a promise with an error
- `cancel` - Cancel a pending promise
- `get` - Retrieve promise details
- `search` - Search for promises by ID pattern and state

## Quick examples

```shell title="Create a promise for human-in-the-loop"
resonate promises create approval-123 --timeout 24h
```

```shell title="Resolve a promise to unblock a workflow"
resonate promises resolve approval-123 --data '{"approved": true}'
```

```shell title="Check promise state"
resonate promises get order-456
```

```shell title="Find all pending promises"
resonate promises search "*" --state pending --limit 10
```

## Learn more

Run `resonate promises --help` to see all subcommands.

Run `resonate promises <subcommand> --help` for detailed usage of each subcommand.
