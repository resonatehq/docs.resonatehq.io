Invoke a function manually from the command line.

```shell title="Invoke a function"
resonate invoke <promise-id> --func <function-name> [flags]
```

Useful for testing functions, triggering workflows manually, or debugging.

## Quick examples

```shell title="Invoke a function with arguments"
resonate invoke order-123 --func processOrder --arg '{"orderId": "123"}'
```

```shell title="Invoke with timeout"
resonate invoke task-456 --func processTask --arg data --timeout 30m
```

```shell title="Invoke with JSON arguments"
resonate invoke job-789 --func analyzeData --json-args '[{"key": "value"}, 42, "text"]'
```

## Common flags

- `--func` - Function name (required)
- `--arg` - Positional argument (repeat for multiple)
- `--json-args` - JSON array of arguments
- `--timeout` - How long to wait for completion (e.g., `30s`, `5m`, `1h`)
- `--target` - Override the default target (e.g., `poll://any@default`)

## Learn more

Run `resonate invoke --help` for all available flags and examples.
