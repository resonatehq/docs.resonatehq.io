Scaffold new projects with the Resonate CLI.

```shell title="Project commands"
resonate projects <subcommand> [flags]
```

## Available subcommands

- `list` - List all available project templates
- `create` - Create a new project from a template

## Quick examples

```shell title="See available templates"
resonate projects list
```

```shell title="Create a new project"
resonate projects create --name my-app --template quickstart-typescript
```

## Learn more

Run `resonate projects --help` to see all subcommands.

Run `resonate projects <subcommand> --help` for detailed usage.

See the [Quickstart guide](/get-started/quickstart) for getting started with new projects.
